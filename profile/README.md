# galua-gameforge

Мы разрабатываем браузерные игры и переиспользуемые TypeScript-библиотеки для
игровых проектов

Основное направление — небольшие веб-игры, которые одинаково удобно работают
на компьютерах и мобильных устройствах и интегрируются с платформой
Яндекс Игры

## Чем мы занимаемся

- создаём браузерные игры на TypeScript и Phaser
- интегрируем рекламу, профиль игрока, облачные сохранения и таблицы лидеров Яндекс Игр
- разрабатываем общие библиотеки для локализации и платформенных интеграций
- автоматизируем проверку, сборку и подготовку релизов
- поддерживаем единые требования к коду и документации во всех проектах

## Проекты

### [2048-bonuses](https://github.com/galua-gameforge/2048-bonuses)

Браузерная версия 2048 с полями разных размеров, бонусами, монетами, магазином,
облачными сохранениями и таблицами лидеров

### [yandex-games-i18n](https://github.com/galua-gameforge/yandex-games-i18n)

TypeScript-библиотека ресурсной локализации веб-игр с типизированными ключами,
fallback-переводами и поддержкой языка из Yandex Games SDK

## Технологии

`TypeScript` · `Phaser` · `Vite` · `Node.js` · `Yandex Games SDK`
